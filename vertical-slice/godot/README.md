# Godot Focus-Mode

Focus-mode vertical slice for _Brew & Justice_.

## Controls

- **F** — toggle focus mode
- **Click** — raise sensory load
- **Hold Space** — rhythmic stim, drops sensory load
- **R** — reset

## Run

Open this project in Godot 4.4.

Open `scenes/focus_mode.tscn` and press **F5**.

## What this scene demonstrates

- Smudge-to-neon clue resolution
- Focus mode dims periphery and boosts perception
- Overload mode fragments audio
- Rhythmic stim reduces sensory load
- Audio bus filters respond to Sensory Meter shifts
- Sensory Meter drives state transitions: Baseline → Hyperfocus → Overload
